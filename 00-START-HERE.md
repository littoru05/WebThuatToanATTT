🔐 6 PHƯƠNG PHÁP MÃ HÓA ĐÃ IMPLEMENT:

1️⃣  CAESAR CIPHER (Mã Hóa Caesar)
    • Loại: Mã hóa cổ điển
    • Input: Shift (0-25)
    • Ví dụ: "Hello" + shift=3 = "Khoor"

2️⃣  VIGENERE CIPHER (Mã Hóa Vigenere)
    • Loại: Mã hóa cổ điển
    • Input: Từ khóa (Key)
    • Ví dụ: "Hello" + key="SECRET" = mã hóa

3️⃣  MODULO ENCRYPTION (Mã Hóa Modulo)
    • Loại: Mã hóa toán học
    • Input: Key + Modulo
    • Ví dụ: Dùng phép modulo để mã hóa

4️⃣  RSA ENCRYPTION (Mã Hóa RSA - Công Khai)
    • Loại: Mã hóa bất đối xứng
    • Input: Cặp khóa công khai/riêng tư
    • Tính năng: Tạo khóa ngẫu nhiên, hỗ trợ 1024/2048 bits

5️⃣  DES ENCRYPTION (Mã Hóa DES)
    • Loại: Mã hóa đối xứng khối
    • Input: Khóa 8 ký tự
    • Lưu ý: Sử dụng chế độ ECB

6️⃣  AES ENCRYPTION (Mã Hóa AES)
    • Loại: Mã hóa đối xứng hiện đại
    • Input: Khóa (bất kỳ độ dài)
    • Lưu ý: Sử dụng chế độ EAX, 256-bit, nonce ngẫu nhiên

🔗 API ENDPOINTS:

POST /api/encrypt
  • Mã hóa dữ liệu
  • Body: {method, plaintext, [method-specific params]}
  • Response: {success: bool, result: string}

POST /api/decrypt
  • Giải mã dữ liệu
  • Body: {method, ciphertext, [method-specific params]}
  • Response: {success: bool, result: string}

POST /api/generate_rsa_keys
  • Tạo cặp khóa RSA
  • Body: {key_size: 1024|2048}
  • Response: {success: bool, keys: {public_key, private_key}}
