╔ HOW IT WORKS ╗

① Generate Keys: Pick 2 large prime numbers (p, q), compute n=p×q, φ(n)=(p-1)×(q-1)

    Choose e coprime to φ(n), compute d = e^(-1) mod φ(n)

② Encrypt: Ciphertext C = Plaintext^e mod n (using public key e, n)

③ Decrypt: Plaintext M = Ciphertext^d mod n (using private key d, n)

💡 Only holder of d can decrypt. Breaking RSA requires factoring n.

📝 Hướng dẫn RSA

  • • Bước 1: Chọn 2 số nguyên tố p, q và số e -> Tạo ra n và khóa riêng d
  • • Bước 2: Nhập văn bản M -> Tính C = M^e mod n
  • • Bước 3: Giải mã -> Tính M = C^d mod n
  • • Ví dụ: p=61, q=53, e=17 → n=3233, d=2753
